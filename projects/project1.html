ABOUT THE PROJECT

Teamed up with a designer to breathe life into a promotional webpage for our beloved show, Adventure Time. This project was a collaborative effort to create an engaging and interactive experience for fans.

The website features a fully responsive design that adapts seamlessly across all devices, from mobile phones to desktop computers. I implemented dynamic content capabilities that allow for easy updates and content management.

One of the key features is the seamlessly integrated newsletter functionality, which keeps fans updated with the latest adventures and news about the show. The implementation ensures a smooth user experience with proper form validation and user feedback.

The project showcases modern front-end development practices, including semantic HTML, CSS Grid and Flexbox for layouts, and vanilla JavaScript for interactivity. All code follows accessibility best practices to ensure the site is usable by everyone.
